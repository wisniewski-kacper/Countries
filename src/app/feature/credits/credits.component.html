Countries app

An application created for the purposes of recruitment.
  • View 0: The list of regions is presented in the form of a side menu. It is available for any view.
  • View 1: The list of views has been prepared in the form of a table. A single record of the table contains the flag of the country, its name and a link to go to view 2 for the country.
  • View 2: Provides some information about the country and shows an image of its flag. Contains a button to return to the list of the region of a given country.
  • About me: the last item in the list launches an author info dialog.
  • Logo: Toolbar text redirects to 'credits' page.
The application uses Angular, NgRx and Material
